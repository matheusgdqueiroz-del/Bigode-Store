Página web de uma loja virtual simples, criada para praticar estrutura de site, estilização com CSS e organização de assets. O projeto inclui página inicial, estilos e imagens para compor a identidade visual.

## Conteúdo

- Página `index.html`.
- Estilos em CSS.
- Imagens e assets da loja.
- Estrutura básica de uma vitrine online.

## Tecnologias

- HTML5
- CSS3
- Git e GitHub

## Como visualizar

Acesse pelo GitHub Pages:

https://matheusgdqueiroz-del.github.io/Bigode-Store/

Ou baixe o repositório e abra o arquivo `index.html` no navegador.
