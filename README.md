Pagina web de uma loja virtual simples, criada para praticar estrutura de site, estilizacao com CSS e organizacao de assets. O projeto inclui pagina inicial, estilos e imagens para compor a identidade visual.

## Conteudo

- Pagina `index.html`.
- Estilos em CSS.
- Imagens e assets da loja.
- Estrutura basica de uma vitrine online.

## Tecnologias

- HTML5
- CSS3
- Git e GitHub

## Como visualizar

Acesse pelo GitHub Pages:

Ou baixe o repositorio e abra o arquivo `index.html` no navegador.
